# scoop-clean-mind

Scoop bucket for [Clean Mind](https://github.com/MS-Teja/clean-mind), a disk
usage analyzer that understands developer bloat.

## Install

```
scoop bucket add clean-mind https://github.com/MS-Teja/scoop-clean-mind
scoop install clean-mind
```

## Update

```
scoop update clean-mind
```

The app is unsigned (no paid code-signing certificate behind it), so Windows
SmartScreen may warn on first run — choose **More info → Run anyway**.
